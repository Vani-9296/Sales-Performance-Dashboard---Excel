# Sales Performance Dashboard (Excel)

## Project Overview

This project demonstrates the creation of an interactive Sales Performance Dashboard using Microsoft Excel. The raw sales dataset was cleaned and transformed using Power Query, analysed with PivotTables, and visualised using Pivot Charts and KPI cards to provide business insights.

---

## Tools Used

- Microsoft Excel
- Power Query
- Pivot Tables
- Pivot Charts
- Excel Formulas

---

## Dataset

The dataset contains sales transaction information including:

- Sales Representative
- Product Category
- Region
- Revenue
- Units Sold
- Customer Satisfaction
- Order Date

---

## Project Workflow

1. Imported the raw sales dataset into Excel.
2. Cleaned and transformed the data using Power Query.
3. Loaded the cleaned data into Excel.
4. Created Pivot Tables for analysis.
5. Built Pivot Charts and KPI cards.
6. Designed an interactive Sales Dashboard.

---

## Dashboard Features

- Total Revenue
- Average Revenue
- Average Unit Price
- Total Transactions
- Customer Satisfaction Score
- Total Units Sold (North Region)
- Sales by Region
- Top 5 Sales Representatives
- Monthly Sales Trend
- Highest Product Category

---

## Key Insights

- North region generated the highest sales revenue.
- Monthly sales showed strong growth toward the end of the year.
- The top-performing sales representative achieved the highest sales volume.
- Product category analysis identified the highest revenue-generating category.
- Total revenue exceeded $3 million across 500 transactions.
